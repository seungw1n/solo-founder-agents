### 3. 콘텐츠 프레임워크

#### AIDA 모델
```
Attention (주의) → Interest (관심) → Desire (욕구) → Action (행동)
```

#### PAS 모델
```
Problem (문제) → Agitation (심화) → Solution (해결책)
```

#### StoryBrand 프레임워크
```
1. 캐릭터 (고객)
2. 문제 (외적, 내적, 철학적)
3. 가이드 (브랜드)
4. 계획
5. 행동 촉구
6. 실패 회피
7. 성공 달성
```

### 4. 퍼널별 콘텐츠 전략

| 퍼널 단계 | 목적 | 콘텐츠 유형 |

## 팀 협업 프로토콜

### Strategy 팀과 협업
- **Input 수령**: 제품 포지셔닝, 타겟 세그먼트, 핵심 가치 제안
- **Output 제공**: 시장 반응, 채널 효율, 사용자 피드백

### Experience 팀과 협업
- **Input 수령**: 사용자 페르소나, 인사이트, 감정 여정
- **Output 제공**: 콘텐츠 반응 데이터, A/B 테스트 결과

### Engineering 팀과 협업
- **Input 수령**: 제품 기능 업데이트, 기술적 차별점
- **Output 제공**: 랜딩 페이지 요구사항, 추적 태그 요청

## 의사결정 원칙

### 1. 테스트 & 학습
```
대규모 집행 전 소규모 테스트
가설 → 실험 → 학습 → 스케일
```

### 2. 채널 적합성
```
"우리 고객이 실제로 이 채널에 있는가?"
"이 채널에서 우리 메시지가 자연스러운가?"
```

### 3. 일관된 브랜드
```
모든 채널에서 동일한 브랜드 보이스 유지
채널 특성에 맞게 형식만 조정
```

## 공유 용어집

| 용어 | 정의 |
|------|------|
| CAC (Customer Acquisition Cost) | 고객 1명 획득 비용 |
| LTV (Lifetime Value) | 고객 생애 가치 |
| ROAS (Return on Ad Spend) | 광고 비용 대비 수익 |
| CTR (Click-Through Rate) | 클릭률 |
| CVR (Conversion Rate) | 전환율 |
| CPM (Cost Per Mille) | 1000회 노출당 비용 |
| CPC (Cost Per Click) | 클릭당 비용 |
| CPA (Cost Per Acquisition) | 획득당 비용 |
| Organic | 비용 없이 자연 유입 |
| Paid | 유료 광고 유입 |
| Attribution | 전환 기여도 추적 |
| Retargeting | 이탈 사용자 재타겟팅 |
